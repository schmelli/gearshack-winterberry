# Implementation Plan: UI/UX Makeover

**Branch**: `006-ui-makeover` | **Date**: 2025-12-05 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/006-ui-makeover/spec.md`

## Summary

Elevate the Loadout Editor and Site Header to premium, consumer-facing quality through layout reversal (picker left, loadout right), enhanced header with Rock Salt title and interactive badges, interactive donut chart with click filtering, gear card images with detail modal, polished site header alignment, and toast feedback system using Sonner.

## Technical Context

**Language/Version**: TypeScript 5.x (strict mode)
**Primary Dependencies**: Next.js 16+ (App Router), React 19+, shadcn/ui, recharts, Sonner (toast), next-themes
**Storage**: Browser localStorage (via existing zustand persist middleware)
**Testing**: Manual testing for MVP
**Target Platform**: Web browsers (Chrome, Safari, Firefox)
**Project Type**: Web application (Next.js App Router)
**Performance Goals**: Toast appears <300ms, chart filter <100ms, mobile sheet <200ms
**Constraints**: Client-side only, must integrate with existing 005-loadout-management components
**Scale/Scope**: UI polish feature, modifies ~10 existing components, adds ~3 new components
